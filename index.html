With our current technology it's already possible to play games more interactive. This project want's to continue the progress of more interactive games and create a tower defense game where one user only can use those towers another user has drawn. A webcam detects and recognizes the towers and gives the permission tu use them in the game.

Focus of this project is the recognition of the towers but the graphics of the game are very important for the appearance, experience and usibility. I can't create high-level graphics so I need your help to hire a graphic designer to develop the game graphics.
